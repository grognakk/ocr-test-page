Step 1: Learn the Basics

HTML tables are created using the following structure:

<table>
    <tr>
        <th>Header 1</th>
        <th>Header 2</th>
    </tr>
    <tr>
        <td>Row 1, Column 1</td>
        <td>Row 1, Column 2</td>
    </tr>
</table>
  • <table> - Creates the table.
  • <tr> - Defines a row.
  • <th> - Defines a header cell.
  • <td> - Defines a standard data cell.
